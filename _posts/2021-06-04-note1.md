# 문자열
```java
  public String sample() {
    return "redirect:/home";
  }
  
  public ModelAndView sample() {
    return new ModelAndView("redirect:/home");
  }
```

# response 객체
```java
  public void sample(HttpServletResponse httpServletResponse) {
    httpServletResponse.sendRedirect("http://google.com");
  }
```

# RedirectView
```java
  public RedirectView sample() {
    RedirectView view = new RedirectView();
    view.setUrl("http://google.com");
    return view;
  }
```

# ResponseEntity
```java
  public ResponseEntity<Object> sample() {
    URI redirectUri = new URI("http://google.com");
    HttpHeaders httpHeaders = new HttpHeaders();
    httpHeaders.setLocation(redirectUri);
    return new ResponseEntity<>(httpHeaders, HttpStatus.SEE_OTHER);
  }
```
